US1 Authentication

US1.1
Sign in with Google Must have
As a user, I want to sign in with my Google account so that the app can access my Google Docs without me entering a password manually.
Acceptance Criteria
  • A "Sign in with Google" button is visible on the first screen
  • Tapping it launches the standard Android Google account picker
  • On successful sign-in, the app navigates to the Drive browser
  • If sign-in fails, a clear error message is shown and the user can retry
PRD: P4.1.1 P4.1.2
US1.2
Sign out Must have
As a user, I want a sign-out option so that I can end my session explicitly and know my data has been cleared.
  • A sign-out button is accessible from the Drive browser screen
  • Tapping it immediately clears all in-memory data
  • The app returns to the login screen
  • No document content or tokens remain after sign-out
PRD: P4.1.4 P6.2
US1.3
Fresh login every launch Must have
As a user, I want the app to always require sign-in on launch so that I know no previous session data is being reused.
  • The app always opens on the login screen, never a previously authenticated state
  • No "remember me" or auto-login feature exists
PRD: P4.1.3 P6.1

US2 Session Management

US2.1
Auto-logout on inactivity Must have
As a user, I want the app to automatically sign me out after 5 minutes of inactivity so that my documents aren't left open if I forget to sign out.
  • A 5-minute countdown starts from the last user interaction
  • Any tap, scroll, or swipe resets the countdown
  • When the timer fires, session data is cleared and the login screen is shown
  • A brief message explains why the session ended
PRD: P4.2.1 P4.2.2
US2.2
Auto-logout when backgrounded Must have
As a user, I want the app to log me out if I switch away from it for more than 5 minutes (without voice reading active) so that my session is protected even if I forget the app is open.
  • Switching to another app starts the inactivity countdown (or continues it)
  • Returning to the app within 5 minutes keeps the session active
  • Returning after 5 minutes shows the login screen
PRD: P4.2.3
US2.3
Voice reading keeps session alive Must have
As a user, I want voice reading to keep my session active so that the app doesn't log me out while I'm listening.
  • The 5-minute inactivity timer is paused while voice reading is playing
  • Stopping voice reading resumes the inactivity timer
  • The sleep timer (US6) governs session end during voice reading
PRD: P4.2.4

US3 Drive Browsing

US3.1
Browse Drive root Must have
As a user, I want to see my Google Drive folders and documents listed after signing in so that I can find what I want to read.
  • Folders appear before documents, both sorted alphabetically
  • Each item shows an icon (folder or document), name, and last modified date
  • Only files owned by me are shown — no shared documents
PRD: P4.3.1 P4.3.6
US3.2
Navigate into folders Must have
As a user, I want to tap a folder and see its contents so that I can find documents organised in subfolders.
  • Tapping a folder loads and shows its contents
  • A breadcrumb shows the current path (e.g. My Drive › Work › Notes)
  • Tapping a breadcrumb item jumps directly to that level
  • Back arrow or system back navigates up one level
PRD: P4.3.2 P4.3.3
US3.3
Search across all docs Must have
As a user, I want to search for a document by name across all folders so that I can find it without knowing where it's stored.
  • A search icon in the toolbar opens a search bar
  • Results show documents from all folders matching the query
  • Each result shows the document name
  • Clearing or cancelling the search returns to the previous folder view
PRD: P4.3.4
US3.4
Open a document Must have
As a user, I want to tap a document and have it open in the reader so I can start reading immediately.
  • Tapping a document opens the reader screen
  • A loading indicator is shown while the document fetches
  • If loading fails, a clear error message is shown with a retry option
PRD: P4.3.5

US4 Document Reading

US4.1
Read with preserved structure Must have
As a user, I want the document to display with its headings, paragraphs, and lists intact so that it's easy to navigate and read.
  • Headings render at visually distinct sizes (H1 largest, H6 smallest)
  • Bullet and numbered lists render with correct indentation and markers
  • Paragraph spacing is comfortable for reading
PRD: P4.4.1
US4.2
Switch between scroll and paginated mode Must have
As a user, I want to switch between continuous scroll and page-by-page reading at any time so that I can choose the style that suits me.
  • A toggle button in the toolbar switches between modes
  • My reading position is preserved when switching
  • In paginated mode, I can swipe left/right or tap edges to turn pages
  • In paginated mode, the current page and total pages are shown
PRD: P4.4.2 P4.4.3 P4.4.4

US5 Voice Reading

US5.1
Start voice reading Must have
As a user, I want to tap a button and have the document read aloud in a natural-sounding voice so that I can listen without looking at the screen.
  • A microphone icon in the reader toolbar opens the voice control panel
  • Tapping Play starts reading from the beginning of the document
  • Audio begins within 3 seconds of tapping Play
  • The voice sounds natural (Google Cloud TTS Neural2)
PRD: P4.5.1 P4.5.7
US5.2
Control playback Must have
As a user, I want play, pause, stop, and skip controls so that I can manage listening without restarting from the beginning.
  • Play / Pause toggles audio on and off
  • Stop ends reading and resets to the beginning
  • Skip forward moves to the next section/chunk
  • Skip back moves to the previous section/chunk
PRD: P4.5.3
US5.3
Adjust reading speed Must have
As a user, I want to control how fast the document is read so that I can listen at a comfortable pace.
  • A speed slider lets me set speed from 0.5× to 2.0×
  • The current speed is displayed numerically (e.g. "1.25×")
  • Speed changes take effect from the next chunk
PRD: P4.5.4
US5.4
Choose a voice Must have
As a user, I want to pick from a list of voices so that I can choose the accent and tone I find most pleasant for long listening sessions.
  • A curated list of Neural2 voices is shown (US, UK, Australian, Indian accents)
  • Each voice shows its name, accent, and gender
  • The selected voice is highlighted
  • Voice changes take effect from the next chunk
PRD: P4.5.2
US5.5
Track reading progress Should have
As a user, I want to see how far through the document the voice reader has got so that I know how much is left.
  • A progress bar shows overall reading progress
  • A section counter shows the current and total sections (e.g. "Section 4 of 12")
PRD: P4.5.5

US6 Sleep Timer

US6.1
Set a sleep timer Must have
As a user, I want to set a sleep timer so that the app stops reading and logs me out automatically when I'm likely to have fallen asleep.
  • A sleep timer button is visible in the voice control panel
  • Tapping it shows options: 15 min, 30 min, 45 min, 60 min, Custom
  • Selecting an option starts the countdown
  • When the timer fires: audio stops, session clears, login screen is shown
PRD: P4.6.1 P4.6.2
US6.2
See timer countdown Must have
As a user, I want to see how much time is left on the sleep timer so that I know when the app will stop.
  • The remaining minutes are shown in the voice control panel while the timer is active
  • It updates every minute
PRD: P4.6.3
US6.3
Cancel the sleep timer Should have
As a user, I want to cancel an active sleep timer so that the app keeps reading if I'm still awake.
  • A Cancel button is visible next to the timer countdown
  • Cancelling removes the timer and the display reverts to the normal sleep timer button
PRD: P4.6.4

US7 Reader Settings

US7.1
Adjust font size Must have
As a user, I want to increase or decrease the font size so that reading is comfortable for my eyes.
  • + and − buttons adjust font size in 1sp steps
  • Range: 12sp to 32sp
  • The current size is displayed between the buttons
  • Heading sizes scale proportionally with body size
PRD: P4.7.1
US7.2
Choose background theme Must have
As a user, I want to switch between light, sepia, and dark backgrounds so that reading is comfortable in any lighting.
  • Three coloured circles represent Light, Sepia, and Dark themes
  • Tapping one applies it immediately to the reader
  • The selected theme is visually indicated
PRD: P4.7.3
US7.3
Choose font family Should have
As a user, I want to pick between serif, sans-serif, and monospace fonts so that I can read in my preferred style.
  • Three font options are shown as text labels
  • Tapping one applies it immediately
  • The selected font is highlighted
PRD: P4.7.2
